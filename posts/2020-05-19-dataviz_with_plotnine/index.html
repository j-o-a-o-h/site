Quando temos interesse em analisar dados e resolver algum problema, logo pensamos nas possíveis tendências e correlações que as variáveis de estudo podem ter. Essa é uma etapa muito importante pois após a formatação e manipulação dos dados brutos, é possível comparar as hipóteses formuladas por nós com as informações que são obtidas por meio de estatísticas e visualizações de dados.

Mas o que é Visualização de Dados?

  • Em resumo, visualização de dados é a representação dos dados em formato gráfico.

Mas o que é um gráfico?

  • Gráfico é uma representação com forma geométrica e que é construída a partir de informações obtidas dos dados. Isso significa que o gráfico é um desenho que tem dimensões, forma, tamanho, comprimento, área, volume, etc.

Essas definições estão bem alinhadas com o livro The Grammar of Graphics de Leland Wilkinson, no qual o autor define um esquema geral para visualização de dados que divide os gráficos em componentes estéticas de objetos geométricos.

A partir desses conceitos, Hadley Wickham escreveu A Layered Grammar of Graphics e criou o ggplot2, um pacote de visualização de dados para a linguagem de programação estatística R. Esse pacote ficou muito popular em diferentes contextos e grupos sociais.

Para a linguagem Python, há o pacote plotnine. Também se baseia em The grammar of graphics e pacote ggplot2 da linguagem R. Portanto, esse post foi feito com o objetivo demonstrar alguns gráficos com a linguagem Python e o pacote plotnine.

1.1 Construção dos primeiros gráficos

A estrutura padrão dos gráficos em plotnine segue a forma:

ggplot(data=<‘DADOS’>, mapping = aes(<‘MAPEAMENTO_VARIAVEIS’>)) + <‘GEOM_FUNCAO’>()

A componente <‘DADOS’> deve ser utilizada para importar os dados (geralmente de um data.frame) Já a componente <‘MAPEAMENTO_VARIAVEIS’> deve ser utilizada para estabelecer se alguma variável irá aparecer no eixo x ou y. Essa também pode ser utilizada por outras variáveis que serão representadas por cores, tamanhos ou formas diferentes. Para selecionar o tipo de gráfico que será plotado, deve-se usar as funções com prefixo geom.

Para a construção do primeiro gráfico, a variável ‘price’ foi escolhida.

g1 = ggplot(data=plotnine_diamonds,mapping=aes(x='price'))
g1.show()

Perceba como essa primeira parte da construção do gráfico não está utilizando a componente <‘GEOM_FUNCAO’>. Somente está sendo delimitado que a variável ‘price’ irá aparecer no eixo ‘x’. Por conta disso, nenhum gráfico foi plotado.

1.2 Histograma

Para a construção de um histograma, basta acrescentar ao mapeamento armazenado no objeto ‘g1’ a função geom_histogram(). No exemplo abaixo, três elementos adicionais foram acrescentados a função geom_histogram() sendo:

  • col: Cor de Contorno
  • fill: Cor de Preenchimento
  • alpha: Quandidade desejada de transparência
g2 = ggplot(data=plotnine_diamonds,mapping=aes(x='price')) + \
        geom_histogram(fill='#CC092F')
g2.show()
g3 = ggplot(data=plotnine_diamonds,mapping=aes(x='price')) + \
  geom_histogram(fill='#CC092F',color='black')
g3.show()
g4 = ggplot(data=plotnine_diamonds,mapping=aes(x='price')) + \
        geom_histogram(fill='#CC092F',color='black',alpha=0.4)
g4.show()

Além desses elementos mencionados, também é possível adicionar outros elementos específicos para cada tipo de geom. Como exemplo, o argumento ‘bins’ do geom_histogram() controla o número de barras será modificado.

g5 = ggplot(data=plotnine_diamonds,mapping=aes(x='price')) + \
        geom_histogram(fill='#CC092F',color='black',alpha=0.4,bins=8)
g5.show()

Como curiosidade, também foi adicionada a função geom_rug(). Essa função basicamente cria linhas abaixo do gráfico para cada observação dentro das faixas de valores definidas pelo histograma.

g6 = ggplot(data=plotnine_diamonds.head(25),mapping=aes(x='price')) + \
   geom_histogram(fill='#CC092F',color='black',alpha=0.4,bins=5) + \
   geom_rug()
g6.show()

1.3 Densidade

Para construir um gráfico de densidade no plotnine, basta adicionar a função geom_density() à estrutura mapeada anteriormente. Novamente, adicionamos a função ‘geom_rug’.

g7 = ggplot(data=plotnine_diamonds,mapping=aes(x='price')) + \
  geom_density(fill='#0059b3') + \
  geom_rug()
g7.show()

1.4 Boxplot

Pensando em análise univariada (uma variável), também é possível criar boxplots pelo pacote.

Aqui o mapeamento foi alterado do eixo ‘x’ (utilizado no histograma e no gráfico de densidade) para o eixo ‘y’. (em ‘x’ talvez seja necessário colocar algum valor).

g81 = ggplot(data=plotnine_diamonds,mapping=aes(y='table')) + \
        geom_boxplot(fill='#0059b3',alpha=0.5)
g82 = ggplot(data=plotnine_diamonds,mapping=aes(x=100, y='table')) + \
        geom_boxplot(fill='#0059b3',alpha=0.5)
g81.show()
g82.show()

Como curiosidade, também foram adicionadas as funções xlab() (Alterar o nome do eixo x), coord_flip() (Girar o gráfico) e ggtitle() (Adicionar um título).

g9 = ggplot(data=plotnine_diamonds,mapping=aes(x=100,y='table')) + \
        geom_boxplot(fill='#0059b3',alpha=0.5) + \
        xlab('table') + \
        coord_flip() + \
        ggtitle('Boxplot')
g9.show()

Quando o assunto é análise bivariada(duas variáveis), podemos adicionar uma nova variável no eixo ‘x’ e fazer um boxplot para cada nível dessa variável.

g10 = ggplot(data=plotnine_diamonds, mapping=aes(x='cut',y='table')) + \
        geom_boxplot(fill='#CC092F',alpha=0.5) + \
        xlab('table') + \
        ggtitle('Boxplots')
g10.show()

Também podemos modificar as cores de contorno ou de preenchimento por essa mesma variável. Essa funcionalidade é mais interessante na presença de mais variáveis, mas fica como exemplo.

g11 = ggplot(data=plotnine_diamonds,mapping=aes(x='cut',y='table',color='cut')) + \
        geom_boxplot(alpha=0.5) + \
        xlab('table') + \
        ggtitle('Boxplots')
g11.show()
g12 = ggplot(data=plotnine_diamonds,mapping=aes(x='cut',y='table',fill='cut')) + \
        geom_boxplot(alpha=0.5) + \
        xlab('table') + \
        ggtitle('Boxplots')
g12.show()

1.5 Gráfico de Barras

Um gráfico de barras pode ser utilizado quando desejamos observar as frequências absolutas ou relativas de uma variável [geom_bar()] ou quando desejamos utilizar a altura das barras para representar outros valores [geom_col()].

Esse tipo de gráfico necessita de um mapeamento com pelo menos duas variáveis, sendo uma no eixo ‘x’ (Variável sendo observada) e outra no eixo ‘y’ (Frequência Absoluta ou Frequência Relativa)

geom_bar()

g13 = ggplot(data=plotnine_diamonds,mapping=aes(x='cut')) + \
       geom_bar(fill='#CC092F',color='black')
g13.show()
  • Altera ordem do eixo x
g14 = ggplot(data=plotnine_diamonds,mapping=aes(x='cut')) + \
       geom_bar(fill='#CC092F',color='black') + \
       scale_x_discrete(limits=plotnine_diamonds['cut'].value_counts().index.tolist())
g14.show()

g22 = ggplot(data=plotnine_mpg,mapping=aes(x='hwy',y='cty',color='displ')) + \
        geom_point()
g22.show()
  • Cria gráfico de dispersão com modificação das cores de preenchimento por outra variável (quantitativa) e acrescenta uma reta de tendência
g23 = ggplot(data=plotnine_mpg,mapping=aes(x='hwy',y='cty',color='displ')) + \
        geom_point() + \
        geom_smooth(method = 'lm',color='blue')
g23.show()
